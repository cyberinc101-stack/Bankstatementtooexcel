Frequently Asked Questions

Is there a limit on how many statements I can convert?
No limit from StatementExcel — convert as many as you need. The only practical limit is your Anthropic API credit balance, which you control yourself.
Does it work with multi-month statements?
Yes. Multi-month PDFs work fine. Very long statements (12+ months) may hit the API token limit and return partial results. For those, split into 3-month chunks for best results.
What happens to my API key?
Your API key is saved only in your browser's localStorage. It is never transmitted to StatementExcel servers. It is only used for direct API calls from your browser to Anthropic.
Can I use this for business bank statements?
Yes — business bank statement PDFs work exactly the same as personal ones. The AI is trained to handle both.
