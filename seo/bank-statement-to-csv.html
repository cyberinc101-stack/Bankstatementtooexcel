Bank Statement PDF to CSV — What You Get

CSV (Comma-Separated Values) is the most universally compatible format for financial data. Every piece of accounting software, spreadsheet application, and fintech tool accepts CSV. Converting your bank statement to CSV means you can use it anywhere.

StatementExcel produces a properly formatted CSV with:

  • UTF-8 BOM encoding so special characters display correctly in Excel
  • Clean column headers: Date, Description, Amount, Balance, Type, Category
  • Proper quoting of description fields to handle commas in merchant names
  • Optional split columns (Withdrawals / Deposits) for accounting software imports
  • Optional totals rows with income and expense summaries

How to Import a Bank Statement CSV into Excel

  1. Download the CSV from StatementExcel
  2. Open Excel and go to File → Open
  3. Browse to the CSV file — Excel detects it automatically
  4. If prompted, select UTF-8 encoding and comma delimiter
  5. Your transactions appear in a formatted spreadsheet

How to Import a Bank Statement CSV into Google Sheets

  1. Download the CSV from StatementExcel
  2. In Google Sheets, go to File → Import
  3. Upload the CSV file
  4. Choose "Comma" as separator and "No" for converting text to numbers
  5. Click Import — done

Importing Bank Statement CSV into Accounting Software

Xero

In Xero, go to Accounting → Bank Accounts → Import a Statement. Upload the CSV. Xero expects columns: Date, Description, Amount (or separate Debit/Credit columns). Use the Split Columns layout in StatementExcel for best compatibility.

MYOB

In MYOB AccountRight, go to Banking → Import Bank Transactions. The Accountant layout from StatementExcel matches MYOB's expected DR/CR format.

QuickBooks

QuickBooks accepts CSV imports via Banking → File Upload. Use the Standard layout — single Amount column with negative values for debits.
