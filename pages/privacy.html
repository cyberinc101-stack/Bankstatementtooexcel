Privacy Policy

Last updated: June 2025

1. Overview

StatementExcel is a browser-based bank statement converter. Your financial documents are never uploaded to StatementExcel servers. All AI processing is done via API calls made directly from your browser to Anthropic using your personal API key.

2. Financial Data

Your bank statement PDFs and their contents are sent directly from your browser to the Anthropic Claude API. StatementExcel does not receive, store, log, or have access to your financial data at any point.

3. API Keys

Your Claude API key is stored in your browser's localStorage only. It is never transmitted to StatementExcel servers. It is used solely for direct API calls from your browser to Anthropic.

4. Analytics

We collect anonymized analytics data via Google Analytics (page views, general usage patterns) to understand how the site is used. This data is not linked to your identity or any documents you process.

5. Advertising

This site displays third-party advertisements via Google AdSense. Ad networks may use cookies for personalized advertising. You can opt out at aboutads.info/choices. We do not control the content of third-party ads.

6. Cookies

We use localStorage to save your API key for convenience. Google Analytics and Google AdSense use cookies for analytics and advertising. See Google's Privacy Policy for details on their data use.

7. Third-Party Services

We use the Anthropic Claude API to process documents. Please review Anthropic's Privacy Policy for information on how they handle API data. Per Anthropic's API terms, data sent via API is not used to train their models.

8. Children's Privacy

This service is not directed at children under 13. We do not knowingly collect personal information from children.

9. Changes

We may update this policy from time to time. Changes will be posted on this page with an updated date.

10. Contact

Questions about this policy can be directed to us via the contact information on our website.
