General

What is StatementExcel?

StatementExcel is a free browser-based tool that converts bank statement PDFs into clean Excel or CSV spreadsheets using AI. It works with any bank, requires no sign-up, and your files never leave your device.

Is it really free?

Yes — StatementExcel itself is completely free. The only cost is the Claude AI API usage billed directly to your Anthropic account. New Anthropic accounts receive free credits worth roughly $5–10, enough to convert dozens of statements.

Do I need to create an account?

No account is needed for StatementExcel. You only need a free Claude API key from console.anthropic.com, which is saved locally in your browser.

Which banks are supported?

All banks that produce text-based PDF statements are supported — Chase, Bank of America, Wells Fargo, Barclays, ANZ, CommBank, and hundreds more worldwide. See the full supported banks list. If your bank isn't listed it almost certainly still works.

Privacy & Security

Is my financial data safe?

Yes. Your PDF is sent directly from your browser to the Anthropic Claude API using your personal API key. StatementExcel never receives, stores, or logs your financial data at any point. We have no server that touches your files.

Where is my API key stored?

Your Claude API key is stored only in your browser's localStorage. It is never sent to StatementExcel servers — only to Anthropic directly when making API calls.

Does Anthropic store my bank statement?

Anthropic processes your document via their API and discards it according to their data retention policies. Review Anthropic's privacy policy for full details.

Using the Tool

What file types are supported?

Text-based PDF files (.pdf) and plain text files (.txt). Most e-statements downloaded directly from your bank's online portal are text-based and will work correctly.

Why did my conversion return 0 transactions?

Your PDF is likely image-based (scanned) rather than text-based. Try selecting text in the PDF — if you can't highlight any, it's a scanned image. Contact your bank and request a digital e-statement instead.

How long does conversion take?

Typically 15–45 seconds depending on statement length and Anthropic API response time. Longer statements with many transactions may take up to 90 seconds.

What is the maximum statement length?

Statements up to approximately 40 pages work reliably. Longer statements may hit API token limits — split them into monthly segments and convert each separately.

What are the 4 Excel layouts?
  • Standard — Single amount column with +/- signs. Best for most users.
  • Split Columns — Separate Withdrawals and Deposits columns (Chase style).
  • Accountant View — Absolute values with DR/CR labels, running balance. Ideal for bookkeeping.
  • Minimal Clean — Date, description, amount only. Best for importing into other tools.
Can I get an Excel .xlsx file instead of CSV?

The download is a CSV file, which opens directly in Microsoft Excel, Google Sheets, and Numbers. CSV is universally compatible with all spreadsheet software. Simply double-click the file to open it in Excel.

Some transactions are missing or wrong — what do I do?

AI extraction achieves 95%+ accuracy but is not perfect. Always verify the extracted data against your original statement before using it for financial or tax purposes. Unusual statement layouts may occasionally cause errors.

I am getting an API error — how do I fix it?
  • Check your API key is correct and starts with sk-ant-
  • Check you have credits remaining at console.anthropic.com
  • Try a shorter statement if the file is very large
  • Wait a moment and try again — occasional API timeouts resolve themselves
