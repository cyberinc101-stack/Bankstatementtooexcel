Step 1: Download Your Bank Statement PDF

Log into your online banking portal and navigate to the Statements or Documents section. Download a monthly or quarterly statement as a PDF. Almost all major banks offer this feature.

Tip: Download the PDF directly from your bank's website rather than printing to PDF from a browser. Bank-generated PDFs are text-based and work perfectly. Browser-printed PDFs are sometimes image-based and cannot be parsed.

Step 2: Get Your API Key

StatementExcel uses the Claude AI API to read your statement. You'll need a free API key:

  1. Visit console.anthropic.com
  2. Sign up for a free account
  3. Go to API Keys and create a new key
  4. Copy the key and paste it into the API Key field on the converter

Your API key is saved locally in your browser — you only need to enter it once. Anthropic provides free API credits to new users, enough to convert dozens of statements.

Step 3: Upload and Convert

Drag your PDF onto the upload area or click to browse. Then click Convert to Excel. The AI will read your statement and extract all transactions — typically in 15–45 seconds depending on statement length.

Step 4: Review and Download

Once complete, you'll see a table of all extracted transactions with:

  • Date — transaction date
  • Description — merchant or transfer description
  • Amount — positive for credits, negative for debits
  • Balance — account balance after transaction (if available)
  • Type — debit or credit
  • Category — auto-assigned category (Groceries, Utilities, Salary, etc.)

Click Download CSV to save the spreadsheet to your computer.

Opening the CSV in Microsoft Excel

  1. Open Excel
  2. Go to File → Open and select your downloaded CSV
  3. Or simply double-click the .csv file — it opens in Excel by default on most systems
  4. All columns will be correctly populated and ready to use
Pro Tip: In Excel, go to Data → From Text/CSV if the file doesn't open correctly. Select "Comma" as the delimiter and "UTF-8" encoding.

Opening in Google Sheets

  1. Open Google Sheets
  2. Go to File → Import
  3. Upload the CSV file
  4. Select "Comma" as separator type
  5. Click Import data

What Happens to My Data?

Your PDF is sent directly from your browser to the Anthropic Claude API using your personal API key. It is not stored by StatementExcel — we never see your data. Anthropic's API processes your document and returns the structured transaction data, then discards the input according to their data retention policies.

Privacy Note: Review Anthropic's privacy policy at anthropic.com/privacy for information on how API inputs are handled. For maximum privacy, consider using a statement with personal information redacted for testing.

Troubleshooting

The conversion returned 0 transactions

Your PDF may be image-based (scanned) rather than text-based. Open the PDF and try to select/highlight text — if you can't, it's image-based. Contact your bank for a text-based digital PDF or request e-statements.

Some transactions are missing or incorrect

AI extraction is highly accurate but not perfect. Review the results against your original statement and correct any errors in Excel before using the data. Unusual formatting or non-standard statement layouts may cause occasional errors.

I'm getting an API error

Check that your API key is correct and that you have credits remaining in your Anthropic account. Very large PDFs (100+ pages) may hit token limits — try converting one month at a time.
